# Forza Fix
**Утилита для устранения вылетов в играх серии Forza Horizon**

![version](https://img.shields.io/badge/version-1.0.1-orange) ![platform](https://img.shields.io/badge/platform-Windows-blue) ![framework](https://img.shields.io/badge/.NET-8.0-purple)

---

## Что это?

Forza Fix — небольшое приложение, которое помогает избавиться от вылетов в играх **Forza Horizon 4 / 5**. Проблема заключается в том, что процесс `explorer.exe` мешает стабильной работе игры и провоцирует краши. Приложение позволяет быстро закрыть и восстановить проводник в один клик. (Данная ошибка была в старых версиях, но у некоторых встречается и в новых)

---

## Возможности

- **Закрыть Explorer** — завершает все процессы `explorer.exe` через Win32 API (`TerminateProcess`)
- **Запустить Explorer** — восстанавливает `explorer.exe` после игровой сессии

---

## Использование

1. Запусти `ForzaFix.exe` **от имени Администратора**
2. Запусти игру
3. Нажми **✕ ЗАКРЫТЬ EXPLORER** — вылеты должны прекратиться
4. После выхода из игры нажми **▶ ЗАПУСТИТЬ EXPLORER** — рабочий стол вернётся в норму

> ⚠️ Без прав администратора закрытие `explorer.exe` может не сработать.
